Final Help Page


There are three kinds of arguements you can use with the program:
  • Boolean value - where you assign the value on the command line.
    • .\final.ps1 -ChangeFile $true
  • String value - you assign a string to a variable to be used in the program. The only one like this is directory
    • .\final.ps1- Directory "c:\MyFolder"
  • Switch value - the value is assumed to be true. If you want a false, you have to do it manually. For example if you want to not do sub directories, you must use do it in the program. Any example would be
    • .\final.ps1 -recurse

Here are the different command line arguements:
  • Directory - used to tell the program what directory to scan. For example:
    • .\final.ps1 -Directory "MyFolder"
  • ViewGui - used to tell the program if you want to see the results in the GUI. It is a switch, and therefore can only be set to true:
    • .\final.ps1 -ViewGui
  • Base - used to tell the program if you want the file that just created to be a base file. There is no real difference between the files, but base files end with _base.csv If you want to run
    you want to run this process and make a base file do:
    • .\final.ps1 -Base
  • Recurse if you want to do sub directories at the same time you do the main directory, use recurse. It will assume you want to do all sub directories below you main directory, so if you use it be careful. If it has lots of files and subdirectories, the program will take a while to finish.
    • .\final.ps1 -Recurse
  • compare - checks if you want to compare it against the base file. The program assumes that you want to use the oldest file is the base file. If there is a different base file you want to use, you will have to do it manually in the program.
    • .\final.ps1 -Compare
  • ChangeFile - checks to see when you compare two files, if you want to make a change file of the differences.Change files end in _changefile.csv. Compare is a switch parameter,
    so by adding it, you are saying yes to compare.
    • finalFinal.ps1 -ChangeFile
  • Help - This screen - The program will exit the program when complete.
    • .\final -Help
  • helpweb - This screen as a web page.
    • .\final.ps1 -helpweb
